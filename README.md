# Ab Shell - A Custom Unix Shell

A lightweight Unix shell implementation written in pure C, inspired by Zsh.

## Features

- ✅ Command line parsing with support for spaces and tabs
- ✅ Built-in commands:
  - `cd [dir]` - Change current directory
  - `pwd` - Print working directory  
  - `exit` / `quit` - Exit the shell
- ✅ External command execution (coming soon)
- ✅ Environment variable management (coming soon)
- ✅ Dynamic memory management

## Requirements

- GCC or any C compiler
- Make
- Unix-based system (Linux/macOS)

## Installation

```bash
# Clone the repository
git clone https://github.com/Abduu898/zsh-clone.git
cd zsh-clone

# Compile the shell
make

# Run the shell
./my_shell
